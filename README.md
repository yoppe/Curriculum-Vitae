## 自己紹介
Webアプリケーションを中心とした開発を５年ほど行っております。

以前の職場でドメイン駆動設計とスクラムを開発に取り入れていたため、仕様の決まりきった物をそのまま作る作業的な開発よりも、
ディレクターの方と密にコミュニケーションを取りながらより価値の高いものを共に作り上げていけるような開発を得意としています。

言語的にはバックエンドはScalaを得意とし、保守性の高いシステムを開発することが得意です。
Scala以外にも最近はPythonをプライベートでも勉強しており、業務でもDjangoを使ったバッチシステムの開発行った経験があります。
フロントエンドはAngular2を始めとするモダンな技術を使った開発が得意としており、SASSなどを使用したUIデザインも行った経験があります。

上記の技術以外でも手続き型のスクリプト言語から関数型のコンパイラ言語まで、幅広い言語経験があり、未経験の言語を使う場合だったとしても
少ない学習コストである程度使いこなせようになるかと思います。また、今まで関わったプロジェクトが新規立ち上げプロジェクトが多かったため、
初期の技術選定などから関われるプロジェクトだとより力を発揮できるかと思います。
