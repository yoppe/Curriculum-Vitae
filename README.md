### 自己紹介

Webサービスを中心とした開発を６年ほど行っており、現在はフリーランスとして活動しております。

以前の職場でドメイン駆動設計とスクラムを開発に取り入れていたため、仕様の決まりきった物をそのまま作る作業的な開発よりも、ディレクターの方と密にコミュニケーションを取りながらより価値の高いものを共に作り上げていけるような開発を得意としています。

言語的にはバックエンドはScalaを得意とし、保守性の高いシステムを開発することが得意です。Scala以外にも最近はPythonをプライベートでも勉強中で、業務でもDjangoを使ったバッチプログラムの開発経験があります。フロントエンドはAngular2を始めとするモダンな技術を使った開発が得意です。またSASSなどを使用したUIデザインやKubernetesの構築/運用、Terraformによるサーバ環境構築の自動化を行った経験もあり、バックエンドからフロントエンド、インフラ周りまで幅広い領域をカバーしています。

上記の技術以外でも手続き型のスクリプト言語から関数型のコンパイラ言語まで、幅広い言語経験があり、未経験の言語を使う場合だったとしても少ない学習コストである程度使いこなせようになるかと思います。また、今まで関わったプロジェクトは新規立ち上げプロジェクトが多かったため、初期の技術選定などから関われるプロジェクトだとより力を発揮できるかと思います。
様々な言語・フレームワーク・ツールを経験する中で習得した、可読性・保守性の高いコードを書く技術に関しては自信を持っております。
